## 📚 学习的技能

### 1. obsidian-markdown
**功能**: Obsidian Flavored Markdown
- Wikilinks: `[[Note Name]]`
- Embeds: `![[embed]]`
- Callouts: `> [!type]`
- Properties: YAML frontmatter

### 2. defuddle
**功能**: 网页内容提取
- 替代 web_fetch
- 去除广告和杂乱内容
- 节省 token

### 3. json-canvas
**功能**: 创建可视化图谱
- Nodes (节点)
- Edges (边)
- Groups (组)
- 可视化知识关联

### 4. obsidian-bases
**功能**: 数据库管理
- Views (视图)
- Filters (过滤器)
- Formulas (公式)

### 5. obsidian-cli
**功能**: Obsidian CLI交互
