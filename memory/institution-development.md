## 已开发的教育助手技能

**技能名称**: edu-assistant.mjs  
**开发时间**: 2026-03-01  
**位置**: ~/.openclaw/workspace/skills/agent-best-practices/scripts/

### 功能模块

| 功能 | 解决的问题 | 预期效果 |
|------|-----------|---------|
| 智能答疑 | 重复问题占用老师时间 | 节省80%咨询时间 |
| 学习规划 | 学生不知道怎么学 | 90天三阶段定制 |
| 进度跟踪 | 学习情况不透明 | 主动干预落后学生 |
| 时政整理 | 资料收集耗时 | 自动生成材料 |
| 试题生成 | 针对性练习不足 | 哪里弱练哪里 |
| 学习提醒 | 学生忘记上课 | 提升到课率 |

### 使用方法
```bash
# 答疑
node edu-assistant.mjs faq "报名时间"

# 规划
node edu-assistant.mjs plan 张三 90

# 跟踪
node edu-assistant.mjs track 张三 8 10

# 出题
node edu-assistant.mjs questions 判断推理 10

# 提醒
node edu-assistant.mjs remind 张三 class
```
